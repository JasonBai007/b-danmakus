# b-danmakus
get danmakus from Bilibili

## How to USE

```
// 安装最新版本python，并且安装前，在弹框下方勾选：将Path添加到环境变量
Install Python 3.x >=3.6 

// 安装第三方模块
pip install bilibili-api

// 执行脚本
python index.py
```
